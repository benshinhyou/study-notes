# 命令行下的Java反编译工具

```shell
curl -O https://www.benf.org/other/cfr/cfr-0.152.jar
# 或者使用 curl -O https://resource.bigcoder.cn/tool/decompiler.jar 

java -jar cfr-0.152.jar Hello.class
```
